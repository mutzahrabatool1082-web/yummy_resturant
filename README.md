Yummy Restaurant Website

📌 Project Overview

Yummy Restaurant Website is a full-stack web application built to provide a smooth and interactive experience for users to explore food items, place online orders, and book tables. The project follows a real-world restaurant management system structure.

🚀 Features

Responsive and modern UI using Bootstrap

Food menu display with attractive layout

Online Order Placement System

Table Booking System

Backend API integration using Node.js & Express.js

Form data handling and storage

Clean separation of frontend and backend
